Working Papers

[1] Mixed Membership Estimation in Two-Path Partial Correlation Network [Job Market Paper] [Available upon request]
Abstract: We propose a methodology to learn the mixed membership in large panel of time series. In contrast to pure membership that one unit exclusively belongs to one group, mixed membership allows for mixed affiliation with several groups. We design a two-path partial correaltion model in which the sparsity structure and the interaction structure are manipulated by mixed membership and sociability structure independently. Different latent structures are integrated into one unified system using Bayesian network and enter into the panel through innovation term. A set of algorithm is proposed to estimate latent mixed membership.

[2] ML-Assisted Empirical Bayes Estimation for Group Regression with Network Data [Paper]
Abstract: We propose a network heterogeneity exposure model which encompasses several network-based panel model as special case. Under the framework, we propose a novel peer effect model based on unit-specific heterogeneity. OLS estimator exists but performs poorly in the presence of sparse network. Then, we propose a machine learning assisted empirical Bayes estimation framework based on (1) structural causal model (2) Bayesian belief network. A specific class of EB estimator is proposed for our network heterogeneity exposure model.

[4] Mixed Membership Estimation in Bayesian Network Autoregression [Paper]
with Endong Wang
Abstract: We propose a network autoregression to learn mixed membership in large panel of time series. The data generating process, coefficient matrix and Mixed membership structure are integrated into one unified system using joint distribution specified by probabilistic graph. We also propose a set of algorithm to learn latent mixed membership structure with the model.

[5] Overlapping Community Detection in Mixed Membership Vector Autoregression [Paper]
Abstract: We propose a mixed membership stochastic block vector autoregression which allows for multiple memberships in large panel of time series. A set of algorithm is proposed to learn the multiple membership structure with the model. We prove the consistency of proposed algorithm.
